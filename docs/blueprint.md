# **App Name**: HARP Insight

## Core Features:

- Data Capture: Capture key HARP details including HARP ID, date, location, department, block, and floor.
- QR Code Generation: Generate QR codes directly from entered HARP data for easy access and sharing.
- PDF Export: Enable exporting HARP data into PDF documents for record-keeping and offline access.
- Location Suggestions: AI-powered suggestions for location, department, block, and floor based on partial inputs and historical data. This is achieved through a tool which incorporates similar details from previous HARP entries in order to give the suggestions.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and security.
- Background color: Light gray (#F0F0F0) for a clean and neutral interface.
- Accent color: Light Blue (#80DEEA) for interactive elements and highlights.
- Body font: 'PT Sans' for clear and readable text.
- Headline font: 'Space Grotesk' for titles and headers.
- Use simple, clear icons to represent actions and data types, in a filled style.
- Maintain a clean, form-based layout consistent with the image, ensuring each data field is clearly labeled.
